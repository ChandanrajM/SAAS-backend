# SaaS Backend - Mini Project Management API

This is a **production-ready backend** for a Mini Project Management SaaS application built with **Node.js, Express, and MongoDB Atlas**.

It includes:

- User registration and login with **JWT authentication**
- Password hashing using **bcrypt**
- Clean folder structure for scalability
- Ready for protected routes and further SaaS features

---

## 🔧 Features

- User authentication (`/api/auth/register` and `/api/auth/login`)
- JWT token-based access for protected routes
- MongoDB Atlas connection
- Environment-based configuration (`.env`)
- Professional folder structure
